Vimium++ Options
Excluded URLs
and keys
Wholly or partially disable Vimium++. A "Pattern" is a regular expression if it starts with "^", or ":" + the header of a URL. Otherwise it will be converted.

If "Keys" is left empty, then Vimium++ will be wholly disabled. Otherwise, just the listed keys are disabled (they are passed through).
Custom key
mappings
Enter commands to remap your keys. Available commands: 
map f LinkHints.activate
unmap <f2>
unmapAll
# this is a comment
" this is also a comment
 Show available commands. .
Custom search
engines
Add search engine shortcuts to the Vomnibar. Format: 
a: http://a.com/?q=$s
js\:|JS: javascript:$S; Run JavaScript
MainKey|c|c\:|MainName: \
  http://c.com/?q=%s&space=\s\ space_end
# this is a comment
 $s will be replaced with your search terms. .
Default search
engine
The default search engine ("~" to use it on Vomnibar)
(e.g. "https://www.baidu.com/s?ie=utf-8&wd=$s Baidu").
New tab URL
"pages/blank.html" for a blank page. Chrome inner:
"chrome-search://local-ntp/local-ntp.html".
Advanced Options
Scroll step size
The size for basic long movements (short = * 0.6).
Characters used
for link hints
The characters placed next to each link after typing "f" to enter link-hint mode.
Vomnibar Page
Your prefered Vomnibar page. Recommended: gdh1995.github.io/vimium-plus/vomnibar.html
Chrome has a memory leak bug with the default page.

Custom CSS
To style Vimium++ nodes, use selectors such as ".S" and ".LH > .MC".

These styles work in addition to and take precedence over Vimium++'s default styles. .
Custom outer CSS
ATTENTION: This feature will be definitely removed in a future release.

These styles are used in addition to and take precedence over all pages' style. .
White list of
other extension ids
Only browser extensions in this list are allowed to request Vimium++'s support.
One extension id per line.
Any line the first char of which is neither a letter nor a number will be ignored. .
Keyboard settings
Your keyboard delay settings on this computer. Format:
[ <delay>, <repeat period> ] (unit: ms)
Locale encoding
In order to recognize urls whose encoding is not UTF-8. The default gbk is for Chinese.
